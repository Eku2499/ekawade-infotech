1. Overview

Ekawade Infotech specializes in providing a wide variety of IT products and services, focused on making daily tasks easy and fast for our clients.

This repository primarily hosts:

The company's marketing website and frontend application.

Service documentation and deployment scripts.

Reusable components and libraries specific to our service offerings.

2. Getting Started

Follow these steps to get a local copy of the project up and running for development or review purposes.

Prerequisites

Ensure you have the following software installed on your machine:

Node.js (LTS Version)

Git

Installation

Clone the repository:

git clone [https://github.com/Eku2499/ekawade-infotech.git](https://github.com/Eku2499/ekawade-infotech.git)
cd ekawade-infotech


Install dependencies:

npm install


or

yarn install


3. Usage

To run the project in a local development environment (with live reloading):

npm start
# or
yarn start


The application will typically be accessible at http://localhost:3000 (check your console output for the exact address).

4. Project Structure

The project follows a standard structure. Key directories are:

Directory

Purpose

/src

Contains all primary source code (components, styles, logic).

/public

Static assets, including the main HTML file, images, and fonts.

/dist

The compiled and bundled output (created after running the build command).

/docs

Documentation for internal processes, services, and architecture.
